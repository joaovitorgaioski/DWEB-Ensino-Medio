Caixas de diálogo

São 3 tipos de caixa de diálogos.

  • alert: Exibe uma mensagem "alert";
  • confirm: Exibe uma mensagem em que o usuário pode escolher entre ok ou cancelar;
  • prompt: O usuário pode entrar dados na caixa de diálogo.

JavaScript pode ser incorporado ao HTML pela tag script. Esse script pode ser interno ou externo ao HTML com um arquivo ".js".
