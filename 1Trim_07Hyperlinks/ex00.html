Hyperlinks

São referências dentro de um documento hipertexto para outras partes do mesmo documento ou outros documentos.

Os links podem ser:

  • Remotos
  • Internos
  • Locais
  • Endereço de correio eletrônico
  • Arquivos para download

Hyperlinks Remotos

São hyperlinks que ligam para outros documentos disponíveis na web. Ex: Clique aqui

Hyperlinks Locais

São hyperlinks hospedados em um mesmo servidor. Ex: Hyperlink local

Hyperlinks em Imagens

É possivel deixar uma imagem clicável, direcionando para uma outra referência.

Hyperlinks Internos

São hiperlinks que referenciam itens dentro do mesmo documento. Ex:

  • Capítulo 1
  • Capítulo 2
  • Capítulo 3
