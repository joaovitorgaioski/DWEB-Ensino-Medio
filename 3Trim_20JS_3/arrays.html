Arrays

Arrays são estruturas de dados que permitem colocar vários dados em uma mesma estrutura com tipos iguais ou diferentes (VETORES).

Faça um programa que seja possível ao usuário inserir frutas em uma lista, item a item através de uma caixa de diálogo. A inserção deve terminar quando o usuário digitar 'fim'. Ao final, a lista deve ser imprimida no docuento HTML. Lembre-se que a palavra 'fim' não deve fazer parte da lista.
