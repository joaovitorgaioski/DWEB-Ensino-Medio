Linha

Para dividi ruma linha horizontal para dividir conteudo, usamos "hr."

Crie um arquivo ex03.html. Coloque um titulo de nivel 1 com o nome "dados pessoais".
Crie um parágrafo para cada um dos campos: "nome, data de nascimento, local de nascimento".
Coloque uma barra horizontal e crie um titulo nivel 2 chamado "endereço".
Na sequência crie um parágrafo e coloque os seguintes campos: "rua, bairro, numero, cidade, cep".
Todos os campos devem ter quebra de linha.
