Como criar tabelas em HTML?

Este é um tutorial com 5 exemplos de como criar tabelas na linguagem de marcação de texto HTML.

Quais tags utilizamos?

Para definir uma tabela:
Usamos a tag "table".
Você pode adicionar atributos para a tabela e personalizar ela.


Para definir a parte que será o cabeçalho da tabela:
Usamos a tag "thead".


Para definir uma linha:
Usamos a tag "tr".
Lembrando que você pode criar quantas linhas você quiser.


Para definir uma coluna DO CABEÇALHO da tabela:
Usamos a tag "th".


Para definirmos a parte que será o corpo da tabela:
Usamos a tag "tbody".


Para definir uma coluna NO CORPO da tabela:
Usamos a tag "td".
Você pode usar os atributos ' rowspan="" ' para juntar colunas e ' colspan="" ' para juntar linhas.

Lembrando que uma coluna necessariamente precisa estar em uma linha, por isso sempre está
acompanhada de uma linha "tr".

Pode-se usar a tag "tfoot" para definir um rodapé para a tabela, mas essa opção esta em desuso.

Colunas também podem ter atributos para sua estilização, como exemplo o atributo ' align="center" '
que centraliza o conteúdo da coluna ou o ' width="" ' que define sua largura.

Resumidamente:

Com esses conhecimentos e os demais a respeito de HTML podemos criar tabelas satisfatórias.
