Desenvolvimento Web :)

Tecnologias Utilizadas:

Front-End

Front-end é o cara que desenvolve o que efetivamente irá aparecer para o cliente.
Ele desenvolve o JavaScript, HTML e CSS.

Back-End

Back-end é o desenvolvedor que fará o processo por trás, aquele que programa o servidor.
Ele programa o banco de dados (Java), configura as transações, Python, etc...

Full-Stack

O desenvolvedor Full-Stack é o cara que domina o Front-End e o Back-End.
Normalmente, empresas pequenas adotam desenvolvidores Full-Stack por parecer mais vantajoso,
porém, a exigencia do desenvolvedor Full-Stack em fazer tudo pode acabar prejudicando a evolução dos sistemas
